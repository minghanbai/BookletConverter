A4轉A6小冊子拼版工具 (Booklet Converter)

核心承諾：檔案不離手 (Client-Side Only)

我們深知您的 PDF 文件可能包含敏感資訊（如個人創作、機密報告等）。因此，本工具採用純前端技術 (Pure Frontend) 開發。

這意味著：您上傳的所有 PDF 檔案、以及重製後的拼版檔案，所有的運算過程完全在您的瀏覽器（您的電腦記憶體）中透過 JavaScript (PDF.js / PDF-Lib) 進行。

我們沒有後端資料庫，也不會將您的任何檔案上傳至雲端伺服器。

資料處理詳情

  • PDF 運算機制： 本工具使用開源的 PDF.js 進行讀取，並使用 pdf-lib 或 canvas 進行影像重製 (Rasterization)。這些操作僅在您開啟網頁的當下於記憶體中執行，關閉網頁後所有資料即消失。
  • 外部服務： 本網站託管於 GitHub Pages。GitHub 可能會收集標準的伺服器存取日誌（如 IP 位址、瀏覽器版本），這屬於 GitHub 的服務條款範疇，與本工具的資料處理無關。此外，我們使用了 Google Fonts 與 CDN 服務來載入字型與程式庫。

關於資安誤判與免責聲明

由於本工具涉及檔案處理 (PDF Processing)，部分資安軟體或瀏覽器保護機制（如 Google Safe Browsing）可能會將含有大量 JavaScript 運算的靜態網頁誤判為風險網站。

若您遇到此類警告，請不用驚慌。這是自動掃描程式的誤報。您可以參考 GitHub 原始碼 來驗證本工具的安全性。
